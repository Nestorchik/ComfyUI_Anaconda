> Example of installation of **ComfyUI** on the **Conda** platform _(Minconda, Anaconda ect.)_
> 
> 在 **Conda** 平台上安装 **ComfyUI** 的示例_（Minconda、Anaconda 等）_

Материал описан в статье на Дзен:
